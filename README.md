## Vim Keys

#### md Template

|-------|-----------------------|
|       |                       |

### Scrolling
| key    | What it does                         |
|--------|--------------------------------------|
| zz     | make line scroll to middle of screen |
| zt     | make line scroll to top of screen    |
| zb     | make line scroll to bottom of screen |
| ctrl-e | scroll one line down                 |
| ctrl-y | scroll one line up                   |
| H      | Scroll to Top                        |
| M      | Scroll to middle                     |
| L      | Scroll to Bottom                     |
